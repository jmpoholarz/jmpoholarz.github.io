upgrade

Spring 2017

Check it out on Github

upgrade is a platform built to connect students with study partners and tutors as a project for my Software Engineering I class. Developed in the Spring semester of 2017, we built this application using Xamarin C# for front-end app design and functionality and JavaScript with the help of Node.js to handle back-end server calls and database manipulation.

Users can create an account and profile and then add themselves to class lists in order to find tutors or classmates to study with. Once connecting with people, users can chat about classes or agree on meeting locations to study.

Although we don't plan to further develop the app, it was a worth the experience in app development and design as well as the teamwork needed to accomplish its development. Looking back on it, we should have better budgeted our time to reduce the crunch before due dates.
