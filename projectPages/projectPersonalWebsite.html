Random Link

Although I previously hosted my personal site on Wix, the performance of the site always seemed to suffer at least somewhat when viewing or editing. Additionally, I had wanted to learn some web development for some time, so remaking everything on my own was an exciting challenge.

So starting with no real web knowledge, I've rewritten my entire personal website using nothing but HTML and CSS. I'm sure I'll find additional ways to improve it over time, but so far, it's been an enjoyable little project in my free time. And it should be much easier to adjust in the future as well seeing as it's just code hosted on Github Pages!
